Cadastro de Animais - Clínica Veterinária

Este projeto permite cadastrar diferentes tipos de animais em uma clínica veterinária. Ele foi desenvolvido em Java, com foco na Programação Orientada a Objetos, e utiliza as seguintes classes:

Arquivos do Projeto

  • Animal.java (Classe mãe abstrata)
  • Cachorro.java (Classe filha)
  • Gato.java (Classe filha)
  • Passaro.java (Classe filha)
  • App.java (Classe principal para executar o programa)

Descrição do Projeto

O sistema permite:

  • O cadastro de três tipos de animais: cachorro, gato e pássaro.
  • O cálculo e exibição de informações dos animais cadastrados.
  • Entrada de dados pelo usuário para registrar os animais.
